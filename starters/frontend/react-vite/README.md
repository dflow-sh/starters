# React (Vite) starter

Minimal React 18 + Vite + TypeScript SPA. Production static output is `dist/` (see `dflow.template.json`).

## Commands

- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Preview production build: `npm run preview -- --host 0.0.0.0 --port 4173`

Deploy manifest: `dflow.template.json`.
